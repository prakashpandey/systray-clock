# Systray-clock

A tiny executable used to display information such as time in system tray.

## Build

- `go get github.com/prakashpandey/x-clock`
- `cd $GOPATH/src/github.com/prakashpandey/x-clock`
- `make build`
- `make run`

## install binary to your class path

- `make install`
